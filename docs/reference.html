Database connection contract

Published packages do not ship SQL drivers. Implement DbConnectionLike with your Postgres/MySQL client:

from snapline.core.types import DbConnectionLike

class AppDb(DbConnectionLike):
    async def query(self, sql: str, params: dict | None = None) -> list[dict]:
        # your pg/mysql driver here
        return rows

Demos use snapline.demo_shared for SQLite and in-memory stubs (create_sqlite_connection, seed_db, db.postgres/mysql/sqlite). See packages/snapline_core/examples/in_memory_db.py for a minimal stub you can copy.

Warehouse comparison

from snapline.core import run_warehouse_comparison, nosql, create_stream_report_writer

await run_warehouse_comparison({
    "sourceDb": source_db,           # DbConnectionLike — your SQL warehouse
    "targetStore": nosql.memory(),   # DocumentStoreLike — e.g. MongoDB adapter
    "tables": [...],                 # WarehouseTableSpec list
    "chunkSize": 50,
    "report": {"outputPath": "./reports/warehouse.jsonl"},
})

create_stream_report_writer appends JSONL events without holding the full dataset in memory.
